Criar um servidor

Criar o arquivo server.js com o código abaixo: 


                            // Importação do módulo http
                                const http = require('http');

                            // Parametros do servidor
                                const host = 'http://localhost';
                                const port = 3000;

                            // Criação do servidor
                                http.createServer( (req, res) => {
                                    res.end('<h1>Working</h1>'); 
                                })

                            // Define um escutador de requisições HTTP para o servidor.
                            .listen(port, () => console.log(`Sever is running in ${host}:${port}`));
Subir o servidor via terminal com o comando node: 

                            node server.js
Testar o servidor criado em http://localhost:3000
