Endpoint para Validar o Token

Criar a rota '/token/validate' para validação de Token em src/routes/authorization.route.ts 


                        authorizationRouter.post(, bearerAuthentication, async (req: Request, res: Response, next: NextFunction) => {
                            try{
                                // Se chegar até aqui, então o Token é valido
                                // Porque já foi autenticado pelo middleware bearerAuthentication
                                res.sendStatus(StatusCodes.OK);
                            }catch(error){
                                next(error);
                            }
                        });
