Programming Paradigm

Introdução

Paradigmas de programação são formas de programar.
Existem dois tipos de paradigmas:

  • Imperativo: Foca em como fazer.
    • Procedural, Orientação a Objeto, Processamento Paralelo.
  • Declarativo: Foca em o quê fazer.
    • Lógico, Funcional, Database.
