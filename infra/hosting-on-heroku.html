hosting-on-heroku

Heroku é uma plataforma de nuvem como serviço que suporta várias linguagens de programação. Free.
Heroku CLI (Command Line Interface) permite criar e gerenciar aplicativos Heroku diretamente do terminal.

  • Inicialização do servidor
    • Usa o script npm start do arquivo package.json para incializar o servidor.
  • Configuração da Porta do Servidor
    • Provê uma porta aleatória e não tem como saber previamente, mas armazena na variável de ambiente process.env.PORT, sendo assim é necessário realizar o tratamento via codificação.
    Exemplo em Node.js 
    
                            .listen(process.env.PORT || port);
  • Configuração do Banco de Dados
    • Usar a variável de ambiente process.env.NODE_ENV para configurar a URI do banco de dados.
  • Configuração do git
    • Usar o arquivo .gitignore para configurar os arquivos que não devem ser enviados para o repositório, e consequentemente para o deploy.
Arquivo Procfile para realizar configurações de ambiente. mLab: Database-as-a-Service for MongoDB
