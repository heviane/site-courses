AWS - Amazon Web Services - Introdução

História

Criada em 2006, a Amazon Web Services (AWS) é uma empresa de serviços de cloud computing, que oferece aos seus clientes acesso a recursos de computação, armazenamento e serviços de hospedagem, como aplicativos, sites, bases de dados, servidores, etc. Fornece também treinamento e certificações.

Introdução

AWS têm origem no Well-Architected Framework.

Pilares

  • Segurança
    • Amplo conjunto de políticas. Tecnologias e controle que fortalecem a segurança, a proteção.
  • Eficiência de Performance
    • Escalabilidade.
  • Confiabilidade
    • Vários servidores, dados espelhados, backup, recuperação dos dados.
  • Excelência Operacional
  • Otimização de Custos
    • Pagar somente pelo uso.
