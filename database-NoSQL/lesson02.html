NoSQL

Tipos de Banco de Dados NoSQL

NoSQL abrange uma grande quantidade de tecnologias, onde a única semelhança entre elas é não serem relacionais.
Essas tecnologias são divididas de acordo com a forma de armazenamento dos dados.

  • MongoDB (Model: Document Store)
  • Redis (Model: Key-Value Store)
  • Cassandra (Model: Wide Column Store)
  • Neo4j (Model: Graph Store)
