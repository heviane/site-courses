GitHub

Introdução

Criado pela Microsoft.
GitHub é um gerenciador de repositórios de arquivos na nuvem. Em sua essência, o GitHub é uma plataforma de colaboração.
Geralmente os usuários usam o Git (software de controle de versionamento), de código aberto e o mais popular atualmente, para rastrear cada contribuição e contribuidor de determinado projeto, assim sabemos exatamente de onde cada linha de código veio.
