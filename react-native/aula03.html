Curso React Native

Criar e configurar App


                        // Criar projeto - Nas últ. versões NPM pode-se usar NPX do Node
                        npx react-native init app01

                        // Lista os emuladores
                        adb devices

                        // Executar App
                        cd app01
                        npm run android
                        npm run start

    Arquivos Gerados

  • __tests__
    • Diretório para os testes automatizados.
  • android e ios
    • Diretório para as compilações para o código nativo.
    O conteúdo (Código Nativo) pode ser aberto no Android Studio e XCode, respectivamente.
  • .buckconfig
  • .editorconfig
  • .eslintrc.js
  • .flowconfig
  • .gitattributes
  • .gitignore
    • Diretórios e arquivos a serem ignorados pelo git.
  • .prettierrc.js
  • Formatador de código.
  • .watchmanconfig
    • Fica olhando as alterações e reflete no emulador.
  • babel.config.js
    • Faz a transpilação para as versões mais antigas do JavaScript.
  • metro.config.js
    • Configurações de compilação.
  • package.json
    • Dependências, Dev Dependências, scripts, etc.
  • package-lock.json ou yarn.lock
  • index.js
    • Ponto de entrada.
  • App.js
    • SPA (Single Page Application) - Componente Root.
  • app.json
    • Configurações do app.
