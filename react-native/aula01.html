React Native

Introdução

Desenvolvido e disponibilizado pelo Facebook, lançado em 2015.

React Native é um framework baseado em React para desenvolvimento multiplataforma para IOS e Android tendo o JavaScript como uma única base de código.
Durante a compilação é realizada a conversão para código nativo do sistema operacional.
O padrão para estilização é Flex Box, portanto não há necessidade de colocar 

display: 'flex';
como fazemos na web.
Os comandos de estilização são identicos aos do CSS, mas usa-se o padrão camelCase, sem hífen.

Vantagens

  • Base de conhecimento compartilhada entre desenvolvimento mobile e frontend
  • É uma ferramenta de mercado, possui comunidade gigante
  • Escalabilidade para o desenvolvimento
  • Facilidade para manutenção
  • Permite liberação de funcionalidades ao mesmo tempo para IOS e Android
  • Flexibilidade para customização

Exemplo básico


                            import React from 'react'
                            import { View, Text } from 'react-native'

                            const App = () => {
                                return (
                                    <View>
                                        <Text>Hello React Native</Text>
                                    </View>
                                )
                            }
                            export default App;
