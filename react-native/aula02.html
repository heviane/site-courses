Ambiente de Desenvolvimento

Windows

    Instalações
  • Chocolatey
    • The Package Manager for Windows.
    Recomendado usar Power Shell.
  • Android Studio
    • Plataforma de desenvolvimento do Android.
    AVD Manager: Aplicativo para gerenciar os dispositivos virtuais.
    Create Virtual Device: Cria um dispositivo virtual.
  • Java 8
  • Python 2
  • Yarn 1
  • Node.js

                            // Instalação via Choco
                            choco install -y nodejs-lts python2 jdk8

                            // Configurações das variáveis de ambiente
                            Computador -> Propriedades -> Config Avançadas Sistema -> Variáveis de ambiente

                                // Criar as variáveis
                                Name: JAVA_HOME
                                Value: C:\Program Files\Java\jdk-16.0.2

                                Name: ANDROID_HOME
                                Value: C:\Android-SDK

                                // Adicionar ao Path
                                %ANDROID_HOME%\emulator
                                %ANDROID_HOME%\tools
                                %ANDROID_HOME%\tools\bin
                                %ANDROID_HOME%\platform-tools
