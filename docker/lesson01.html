Há uma camada de Proxy/Service antes de chegar nos containeres.
Pode-se ter vários conteineres espalhados, facilitando a substituição de um contêiner por outro.
O ambiente de update é possível de uma maneira simples usando o Docker Swarm e o Kubernetes.
Podemos usar o Docker Compose para fazer os updates.
