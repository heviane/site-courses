Client-side

Execução no browser via Google Chrome Developer Tools.
Basta digitar o código na aba console.

OBS: Cada browser tem a sua versão do Developer Tools.

Server-side

Execução no terminal via Node.js.
O Node deve ser instalaldo nã máquina onde se pretende executar.
No terminal, navegue até o diretório onde encontra-se o arquivo .js contendo código JavaScript
Execute o comando node seguido pelo nome e extensão do arquivo: 

                            node script.js

Módulos (Importação e Exportação)

É possível importar e exportar módulos para que possamos utilizar as funções e variáveis de outros arquivos.

ES5 - Sintaxe CommonJS


                            // Exportação
                            module.exports = {
                                foo: function(){ return 'bar' },
                                baz: 123
                            }

                            // Importação
                            const A = require('./A.js')
                            const foo = A.foo;
                            const baz = A.baz;

ES6 - Sintaxe Import


                            // Exportação
                            export function foo(){ return 'baz' };
                            export const baz = 123;
                            
                            // Importação
                            import * as A from './A.js';
                            const foo = A.foo;
                            const baz = A.baz;
