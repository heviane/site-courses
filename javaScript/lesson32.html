DOM - Selecionar Elementos

Podemos acessar os elementos html através do objeto document e seus métodos.

Há quatro formas (parametros) para selecionar os elementos:

  • id: É único, só podemos ter um
  • classe: Array HTMLCollection
  • tag: Array HTMLCollection
  • name: Array HTMLCollection

Método document.getElement...


                            // pelo atributo id (único)
                                document.getElementById('name');

                            // pelo atributo class
                                document.getElementsByClassName('name');
                                document.getElementsByClassName('name')[0];

                            // pelo nome da tag
                                document.getElementsByTagName('p');
                                document.getElementsByTagName('p')[0];

                            // pelo atributo name
                                document.getElementsByName('phone');

Método document.querySelector


                            // pelo atributo id (único)
                                document.querySelector('#name');

                            // pelo atributo class
                                document.querySelector('.name');       // retorna o primeiro
                                document.querySelectorAll('.name');    // retorna um array
                                document.querySelectorAll('.name')[0]; // retorna um de acordo com o index

                            // pelo nome da tag
                                document.querySelectorAll('li')[0];
                                document.querySelectorAll('li')[1];

                            // pelo atributo name
                                document.querySelector('age');
