Como executar o JavaScript

Console na web e no terminal localmente

Client-side

Execução no browser via Google Chrome Developer Tools.
Basta digitar o código na aba console.

OBS: Cada browser tem a sua versão do Developer Tools.

Server-side

Execução no terminal via Node.js.
O Node deve ser instalaldo nã máquina onde se pretende executar.
No terminal, navegue até o diretório onde encontra-se o arquivo .js contendo código JavaScript
Execute o comando node seguido pelo nome e extensão do arquivo: 

                            node script.js
