Regex (Regular expression)

Introdução

Expressões regulares são padrões utilizados para selecionar combinações de caracteres em uma string.
Desta forma, podemos buscar, validar e alterar qualquer padrão de caracteres em qualquer texto.

Regex no Javascript

Em JavaScript, expressões regulares também são objetos.
Elas podem ser utilizadas com os métodos exec() e test() do objeto RegExp, e com os métodos match(), replace(), search(), e split() do objeto String.
