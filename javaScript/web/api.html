API (Application Programming Interface)

O que é API?

É uma forma de intermediar o backend com o frontend.
O acesso ocorre por meio de uma URL.
JSON (Javascript Object Notation) é uma forma de armazenar e transmitir dados.
Este é um dos formatos mais usados em APIs para receber e/ou enviar dados na web.

Como consumir API?

A forma mais simples de consumir requisições HTTP em javascript é através da API Fetch do browser
