Input e output de dados com JavaScript

Importante distinguir:

  • window: Janela do navegador
  • document: Página html
  • console: Console de Depuração
    • Navegador: Developer Tools
    • Node: Terminal Node

Entrada de Dados

Abre prompt para digitação. 

                    const text = prompt("Digite um texto.");
                    alert('Texto Digitado: ' + text);

Visualização de Dados

Exibe no prompt. 

                    window.alert('Hello 1'); 
                    alert('Hello 2');
Exibe no body do html. 

                    document.write('Hello');

Impressão de Dados

Abre prompt para impressora. 

                    print('Imprimir dados...');
