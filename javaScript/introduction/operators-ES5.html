Operadores

Operadores são sinais utilizados para manipular dados ou validar condições.

Tipos de Operadores

  • Unário (Recebe 1 operando e pode vir antes ou depois do operador)
    • 
                                ++, --, !, !!, -, +, ~, typeof, void, delete
  • Binário (Recebe 2 operandos e o operador vai no meio)
    • 
                                +, -, *, /, %, **, <<, >>, >>>, &, |, ^, in, instanceof
  • Ternário (Recebe 3 operandos)
    • 
                                ?:

Operadores

  • Atribuição
    • 
                                =, +=, -=, *=, /=, %=
  • Aritmético: São operadores matemáticos.
    • 
                                +, -, *, /, **, %, ++, --
  • Comparação: Retorna boolean (true ou false).
    • 
                                ==, ===, !=, !==, <, >, <==, >==
  • Lógico: Retorna boolean (true ou false).
    • 
                                &&, ||, !
  • Condicional: if ternário.
    • value1 se condição verdadeira, value2 se condição falsa. 
    
                                // Ternário
                                    (condition) ? value1 : value2;

                                // Equivalente a:
                                    if (condition) {
                                        value1
                                    } else {
                                        value2
                                    }
