Executar código JavaScript

Client-side

Execução no browser via Developer Tools (Cada browser tem o seu).
Basta digitar o código na aba console.

Server-side

Execução no terminal via Node.
O Node deve ser instalaldo nã máquina onde se pretende executar.
No terminal, navegue até o diretório onde encontra-se o arquivo .js contendo o código JavaScript.
Execute o comando node seguido pelo nome e extensão do arquivo: 

                            node script.js
