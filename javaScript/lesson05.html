Variáveis, Escopo, Tipos de Dados

Tipos de Variáveis e seus escopos

  • var
    • Escopo global e local (de função). Não entende escopo de bloco.
    Valor inicial é opcional, o default é undefined. Mutável.
  • let
    • Escopo de bloco (ES6).
    Valor inicial é opcional, o default é undefined. Mutável.
  • const
    • Escopo de bloco (ES6).
    Valor inicial é obrigatório.
    Imutável.
    • tipo primitivo: É imutável, não permite alterar valor.
    • tipo referência: É imutável, não permite alterar referência.
      • Porém, permiti alterar propriedades (Objects e Arrays).
      Objetos possuem um tipo de configuração que permite bloquear este tipo de permissão.
    BOA PRÁTICA: Escrever constantes em letras maiúsculas.

Closure - Escopo Léxico

Capacidade de uma variável de lembrar do ambiente onde foi criada.

  • Escopo global
    • No browser o escopo global é representado por "window".

  • Escopo de função
    • 
                                function fn(){
                                    // Escopo de função
                                }
  • Escopo de bloco
    • É definido por um par de chaves {}. ES6 com a inclusão let e const 
    
                                {
                                    // Escopo de bloco
                                }

    Boas Práticas:

  • Não criar variáveis globais.
  • Sempre trabalhar com funções puras.
    • Função pura: Recebe um parâmetro (assim, sabemos de onde está vindo) e retorna alguma coisa.

Tipagem Fraca e Dinâmica

  • Tipagem fraca
    • Permite realizar operações com tipos diferentes sem dar erro, pois resolve por ele mesmo.
  • Tipagem dinâmica
    • Não há necessidade de explicitar o tipo no momento da declaração.
    Inferência de tipo: O compilador, em tempo de execução, atribui automaticamente de acordo com o valor atribuído.

Tipos de Dados Primitivos e Referência

Tipo de dado primitivo

  • string: Representa texto
    • Não confundir com o tipo String que é um objeto do JavaScript, ou seja, um tipo de dado de referência.
  • number: Representa todos os números (positivos, negativos, com ou sem casas decimais)
  • boolean: Representa true ou false
  • null
  • undefinied: Variável declarada, mas não atribuida (sem um valor definido)
  • symbol: As instâncias são únicas e imutáveis (ES6)

    • Tipo de dado referência

  • Object: Lista nomeada. É um Lietral Object
  • Array: Lista numerada. É um Iterable Object
  • Function: Permite ser chamada (Called). Também é um objeto
