JavaScript

Introdução

JavaScript (frequentemente abreviado como JS) é uma linguagem de programação de alto nível baseada em script conforme a especificação ECMAScript. Foi criado em 1995, por Brendan Eich, e lançado junto com versão beta do navegador netscape.

Juntamente com HTML e CSS, o JavaScript é uma das três principais tecnologias da World Wide Web, servindo para criar páginas dinâmicas e iterativas, devido a sua capacidade de interagir com elementos html via DOM, sendo assim uma parte essencial dos aplicativos da web. Todos os navegadores têm um mecanismo dedicado a executar JS.

Vanilla JS nada mais é do que JavaScript puro.

Principais Características

  • Interpretada: Execução em tempo real, não gera código intermediário, geralmente compilado just-in-time
  • Multiparadigma: Estruturado, funcional, orientação à objeto
  • Baseada em Protótipos: Base (conjunto de funções) para estrutura de dados
  • Padrão ECMAScript: Language Specification
  • Linguagem da Web: Nasceu client-side (Interpretada por todos os navegadores)
    • Atualmente é server-side também (Node.js)
  • Tipagem dinâmica e fraca

Evolução e Timeline

Todo ano tem atualizações do ECMAScript, esses são os anos mais importantes:
  • 1995: Criado
  • 1997: ECMAScript 1 (First edition) JS começou a usar este padrão!
  • 1998: ECMAScript 2 (Editorial changes only)
  • 1999: ECMAScript 3 (Regular Expressions, try/catch)
  • 2000: ECMAScript 4 (Never Released)
  • 2009: ECMAScript 5
    • "strict mode", JSON support, String.trim(), Array.isArray(), Array Iteration Methods.
  • 2015: ECMAScript 6
    • Muita atualização...
