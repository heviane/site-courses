Throw

A instrução throw lança uma exceção definida pelo usuário.
Sintaxe: 


                    throw expression; // lança uma exceção
O lançamento da exceção interrompe a execução do código, as instruções posteriores ao throw não serão executadas e o controle será passado para o primeiro bloco catch encontrado na pilha de chamadas, se nenhum bloco catch existir a execução é encerrada. 

                    console.log('before the error');
                    throw 'throwing error...';  // execução encerrada aqui...
                    console.log('after the error');

                    console.log('before the error');
                    throw new Error('throwing error...'); // execução encerrada aqui...
                    console.log('after the error');
