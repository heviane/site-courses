Keyed collections: Set Object

São coleções que armazenam valores únicos.
Cada valor só pode ocorrer uma vez em um Set.
Pode conter qualquer valor de qualquer tipo de dados.

Métodos


                        const myArray = [1,2,3,4,5];

                        // Declaração
                        const mySet = new Set(myArray);

                        // Método add() para Adicionar
                        mySet.add(6); //

                        // Método has() para verificar se valor existe
                        mySet.has(1); // true

                        // Método delete() para Remover
                        mySet.delete(4);

Diferença entre Set e Array

Set

  • Possui valore únicos
  • Propriedade size (Equivalente a length)
  • Não possui os métodos map(), filter(), reduce(), etc.
    • Tem menos operações que os arrays.
