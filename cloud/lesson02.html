AWS - Amazon Web Services - Projeto

Arquitetura

  • Criar um pool de usuários com Amazon Cognito
  • Criar uma tabela com DynamoDB e criar funções Lambda
    • Sempre que trabalhar com Lambda precisa configurar permissões no IAM.
  • Criar uma API Gateway e rotas
  • Desenvolver as integrações
  • Configurar permissões em IAM (Identify and Access Management)
    • Configurar polices e roles.

Serviços Usados para Realizar Deploy da App na AWS

  • AWS RDS - Serviço de Gerenciamento de Banco de Dados
    • Suporta uma gama de BD Relacionais, é nele que cria-se as instâncias.
  • AWS ECR - Repositório das imagens dos containeres
    • Outra solução seria o Docker Hub.
  • AWS ECS - Orquestrador de containeres
    • Outra solução seria o Kubernetes (Open Source).
  • Github Actions - Fazer integração do GitHub com a AWS
