React

Introdução

História

É uma biblioteca JavaScript para construir interfaces de usuário.
Criada em 2011 pelo facebook.
Baseada no XHP, framework para criação de HTML no PHP.

Vantagens

  • Maior controle (Não fica amarrado, dependente de um framework e a evolução dele).
  • Trabalhar com JavaScript de forma mais simples e rápida.
  • É umas bibliotecas mais usadas atualmente. Comunidade grande.

Desvantagens

Ecossistema é muito grande (Muitas ferramentas, nem todas são muito úteis).

Onde usar?

  • Frontend web
  • Frontend mobile com React Native **2015
  • Backend com server side render no NextJS
  • Desktop com UWP (Universal Windows Platform) **2015

Conceitos

React usa linguagem declarativa

Imperativa: Preocupa-se com passos a serem tomados para determinado resultado.
Declarativa: Preocupa-se apenas em exibir ou não, não importam condições, passos.

SPA - Single Page Application

É uma página de aplicação única.
Dentro dessa página é organizado, controlado e renderizado todas as rotas.
Todas as rotas executarão sempre dentro desta página única.
Isso faz com que a sessão permaneça sempre no mesmo contexto.
O browser recebe a página única e renderiza de uma única vez todas as rotas configuradas nela.

Vantagens

  • Melhor performance
  • Melhor entendimento do código
