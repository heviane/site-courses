Props

Props é um objeto que contém as propriedades (variáveis e funções) de um componente.
Podemos usar as props das seguintes maneiras:

  • Acessando diretamente determinada propriedade
    • 
                                props.name;
                                props.value;
                                props.onClick();
  • Desestruturação da props
    • No corpo da função, antes do return:
    
                                    const { name, value, onClick } = props;
    Na lista de argumentos da função.
    
                                    const componentA = ( { name, value, onClick } ) => {...}
    Prop drilling é um estágio do desenvolvimento que acontece quando precisamos obter dados que estão em várias camadas na árvore de componente react.
    Podemos passar props de um componente para outro, acessar as props de um componente filho através de um componente pai.
    https://www.alura.com.br/artigos/prop-drilling-no-react-js props.children é uma propriedade de props para trabalhar com componentes aninhados.
    
                            import React from 'react'; 
                            function ComponentA(props){
                                return(
                                    <div>Component A
                                        <div> { props.children } </div>
                                    </div>
                                );
                            }
                            export default ComponentA;
