Curso de React.js

Ecossistemas do React

React funciona sozinho, mas para apps robustas e grandes projetos se faz necessário o uso de ferramentas auxiliares.

Exemplos

  • React Router
  • Redux
    • Para gerenciamento de estado, mas com Hooks dá para usar outras abordagens.
  • Material UI
    • Para criação de interfaces, estilos prontos.
  • Ant-Design
    • Idem Material UI.
  • Storybook
    • Apresentação de componentes visuais em modo de desenvolvimento.
  • Gatsby
    • Gerenciador de sites estáticos.
  • Jest
    • Para testes unitários.
  • React i18n Next
    • Para internacionalização de textos.
