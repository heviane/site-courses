MSW - Biblioteca para mockar requisições

O objetivo é mockar o retorno do servidor, para ser sempre o mesmo, assim podemos testar o código e este sempre passará nos testes, sem a necessidade de precisar levantar um servidor real.

SW (Service Worker): É uma tecnologia usada pelos browsers que serve para, se cair na camada de rede, pode-se fazer cache (files, requisitions, etc).
Neste caso, está sendo usada para mockar requisições. 


						import { rest } from 'msw';    // Biblioteca          
						import { setupServer } from 'msw/node';  // Ambiente é node
