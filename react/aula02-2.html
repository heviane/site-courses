React Scripts

This package includes scripts and configuration used by Create React App.
Comando para instalar React Scripts:

                            $ npm i react-scripts
Ou podemos configurar com Task Runners e Bundler Sizers (Parcel, Webpack, etc).
O react scripts já instala tudo que precisa e já monta e configura todo o ambiente, como: servidor, webpacks (transpilador Babel), JEST para testes.
Task Runners são executores de tarefas.
Bundle Size An app's bundle size is the amount of JavaScript a user will have to download to load your app.

Prática: Criar uma app usando React Scripts

Create a new React-based project using create-react-app:

                        $ npm init react-app ./my-react-app
                        $ cd my-react-app
                        $ npm install --save react@16.8.6 react-dom@16.8.6 react-scripts@3.0.1
                        $ npm start
package.json é o arquivo de configuração do projeto (contém todas as dependências).
manifest.json provides metadata used when your web app is installed on a user's mobile device or desktop.

Outros comandos úteis

npm start Para começar o desenvolvimento.
npm run build Para criar um pacote de produção.
Aprenda como configurar uma URL pública não root executando `npm run build`.
npm test Iniciar o executor de teste.
npm run eject Remove esta ferramenta e copia dependências de construção, arquivos de configuração e scripts para o diretório do aplicativo. Se você fizer isso, você não pode voltar!
