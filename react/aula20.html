Tratamento de Erros

    Benefícios

  • Resiliência de software Quando falhar, o que mostrar?
  • Segurança
Exemplo de tratamento de erro 

                        // Antes
                        export const sum = (a, b) => a + b;

                        // Depois
                        export const sum = (a, b) => {
                            if (typeof a === number && typeof b === number) {
                                return a + b;
                            } else {
                                // vamos convencionar -1 quando a soma for inválida
                                return -1;
                            }
                        };
    Outros exemplos de tratamento de erros:
  • Validação de formulários
  • Validação de dados (Obrigatoriedade, mascara, tipos de dado, etc)
  • Seguança (uso de captcher para evitar robots)
  • Tratamento por sanitização (de dados sensíveis)
  • Mensagens de alertas e erros (Para orientar)
  • Tratar erro do servidor para erro mais amigável no frontend
  • Segurança (Não expôr App)
    Tratamento de erros em componentes React:
  • Em JS ProTypes
  • Linguagens tipadas como TypeScript, definindo interfaces
Exemplo tratamento de erro em componente 

                        import React from 'react';
                        import PropTypes from 'prop-types';

                        export const Basic = ( { name } ) => ( <p>Meu nome é { name }</p> );

                        Basic.propTypes = {
                            name: PropTypes.string
                        };
