Debugging

É o processo de depuração para encontrar e reduzir defeitos de programação.

    Formas

  • Manual
  • Logging
  • Ferramentas especificas
    • Chrome ou Mozila DevTools
    • React DevTools
    • Redux DevTools

DICA: Escrever a palavra reservada debugger na linha do código que deseja depurar. Desta forma, no Chrome DevTools, pode-se executar o código passo-a-passo e ir visualizando a stack realcionada ao lado.
