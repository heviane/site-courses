DQL - Data Query Language

Linguagem para consulta de dados.
Obs: Vale destacar que algumas literaturas colocam o SELECT como integrante do grupo DML, já que a sua função nesse grupo seria de recuperação de dados.

SELECT


                            SELECT * FROM table_name;
                            SELECT id, name, email FROM user;

WHERE


                            SELECT * FROM user WHERE id = 1;
                            SELECT * FROM user WHERE name = 'Heviane';

                            SELECT * FROM user WHERE name LIKE '%Hevi%';
                            SELECT * FROM user WHERE name LIKE '%Hevi';
                            SELECT * FROM user WHERE name LIKE 'Hevi%';
