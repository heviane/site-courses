SQL - Structured Query Language

    What is SQL?

  • SQL stands for Structured Query Language (Linguagem de consulta estruturada).
  • SQL is designed to manipulate data in Relational Database Management System (RDBMS).
  • Programming paradigm: Declarative.
  • Family: Query language
  • Filename Extension: .sql
  • Internet media type: application/sql
  • Standard: ISO/IEC 9075
Convenção: Usar letras maiusculas para comandos e palavras reservadas.

Comandos são divididos em conjuntos.A divisão realizada de acordo com o que cada comando faz, comandos com atividades similares são agrupados no mesmo conjunto.

    Commands

  • DDL - Data Definition Language
  • DML - Data Manipulation Language
  • DQL - Data Query Language
  • DTL - Data Transaction Language
  • DCL - Data Control Language
